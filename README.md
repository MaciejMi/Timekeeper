## Run Locally

To run Timekeeper locally, run this command on your git bash:

Linux and macOS:
```bash
sudo git clone https://github.com/MaciejMi/Timekeeper.git
```

Windows:
```bash
git clone https://github.com/MaciejMi/Timekeeper.git
```
